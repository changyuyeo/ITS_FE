# Spinner

&lt;Spinner /&gt; 컴포넌트 사용법 정리

```ts
// Spinner 타입 정의
interface SpinnerProps {
	color?: 'primary' | 'secondary' | 'success' | 'error' | 'white'; // default: 'primary'
	duration?: number; // default: 500
	isAbsolute?: boolean; // default: false
	isFixed?: boolean; // default: false
	size?: 'sm' | 'base' | 'lg'; // default: 'base'
	zIndex?: number; // default: 999
}
```
<br />

## 기본 사용법

- `Spinner`컴포넌트는 로딩 중인 상태를 표현할 때 유용합니다.

<Canvas>
	<Story name="basic">
		<Spinner />
	</Story>
</Canvas>

<br />

## color

- `color` 속성을 이용해서 Spinner 컴포넌트의 색상을 변경할 수 있습니다.

<Canvas>
	<Story name="color">
		<Spinner color="primary" />
		<Spinner color="secondary" />
		<Spinner color="success" />
		<Spinner color="error" />
		<div style={{ background: '#333', padding: '8px' }}>
			<Spinner color="white" />
		</div>
	</Story>
</Canvas>

<br />

## size

- `size` 속성을 이용해서 Spinner 컴포넌트의 크기로 설정할 수 있습니다.

<Canvas>
	<Story name="size">
		<Spinner size="sm" />
		<Spinner size="base" />
		<Spinner size="lg" />
	</Story>
</Canvas>

<br />

## duration

- `duration` 속성을 이용해서 Spinner의 속도를 조절할 수 있습니다.

<Canvas>
	<Story name="duration">
		<Spinner duration={200} />
		<Spinner duration={300} />
		<Spinner duration={1000} />
		<Spinner duration={7000} />
		<Spinner duration={10000} />
	</Story>
</Canvas>

<br />

## isAbsolute, isFixed

- `isAbsolute` 속성은 Spinner의 포지션을 absolute로 변경해 주는 속성이며,
- `isFixed` 속성은 포지션을 fixed로 변경해주는 속성입니다.

<Canvas>
	<Story name="isAbsolute, isFixed">
		<Spinner isAbsolute />
	</Story>
</Canvas>

<br />

## zIndex

- `zIndex` 속성은 Spinner의 z-index 값을 직접 설정할 수 있는 속성입니다.

<Canvas>
	<Story name="zIndex">
		<Spinner zIndex={1} />
		<Spinner zIndex={10} />
		<Spinner zIndex={100} />
		<Spinner zIndex={999} />
	</Story>
</Canvas>
