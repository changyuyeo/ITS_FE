# Button

&lt;Button /&gt; 컴포넌트 사용법 정리

```ts
// Button 타입 정의
interface ButtonProps {
	children: ReactNode; // required value
	disabled?: boolean; // default: false
	fullSize?: boolean; // default: false
	htmlType?: "button" | "reset" | "submit"; // default: "button"
	icon?: ReactNode;
	loading?: boolean; // default: false
	outline?: boolean; // default: false
	shape?: "circle" | "default"; // default: 'default'
	size?: "sm" | "base" | "lg"; // default: 'base'
	type?: 'primary' | 'secondary' | 'success' | 'error' | 'gray' | 'dark'; // default: 'primary'
}
```
<br />

## 기본 사용법

- `Button`컴포넌트는 기본적으로 **children**을 받습니다.

<Canvas>
	<Story name="basic">
		<Button>Button</Button>
	</Story>
</Canvas>

<br />

## type

- `type` 속성을 이용해서 Button 컴포넌트의 색상을 변경할 수 있습니다.

<Canvas>
	<Story name="type">
		<Button type="primary">Button</Button>
		<Button type="secondary">Button</Button>
		<Button type="success">Button</Button>
		<Button type="error">Button</Button>
		<Button type="gray">Button</Button>
		<Button type="dark">Button</Button>
	</Story>
</Canvas>

### 참고!

- 기본 html 에서 제공하는 type prop은 `htmlType` 이라는 속성으로 대체하게 됩니다. 

<Canvas>
	<Story name="htmlType">
		<Button htmlType="button">Button</Button>
		<Button htmlType="reset">Button</Button>
		<Button htmlType="submit">Button</Button>
	</Story>
</Canvas>

## outline

- `outline` 속성을 이용하면 해당 색상의 테투리를 가지게되며 색상은 하얀색으로 변경됩니다.

<Canvas>
	<Story name="outline">
		<Button outline type="primary">Button</Button>
		<Button outline type="secondary">Button</Button>
		<Button outline type="success">Button</Button>
		<Button outline type="error">Button</Button>
		<Button outline type="gray">Button</Button>
		<Button outline type="dark">Button</Button>
	</Story>
</Canvas>

<br />
