# Textarea

&lt;Textarea /&gt; 컴포넌트 사용법 정리

```ts
// Textarea 타입 정의
interface TextareaProps {
	autoSize?: boolean; // default: false
	resize?: boolean; // default: false
	suffix?: ReactNode;
}
```
<br />

## autoSize

- `autoSize` 속성을 이용해서 Textarea 컴포넌트의 크기로 설정할 수 있습니다.
- 고정 높이가 아닌 textarea 컨텐츠 내용에 따라서 높이가 자동으로 조절됩니다.

<Canvas>
	<Story name="autoSize">
		<Textarea autoSize placeholder="값을 입력해주세요." />
	</Story>
</Canvas>

<br />

## resize

- `resize`를 이용해서 사용자가 임의 사이즈를 변경 가능하도록 설정합니다.

<Canvas>
	<Story name="resize">
		<Textarea resize rows={5} placeholder="값을 입력해주세요." />
	</Story>
</Canvas>

<br />

## suffix

- `suffix`가 발생할 때 Textarea 컴포넌트에서 에러 애니메이션을 발생합니다.

<Canvas>
	<Story name="suffix">
		<Textarea suffix={<XCircle fill="#777" />} placeholder="값을 입력해주세요."  />
	</Story>
</Canvas>
