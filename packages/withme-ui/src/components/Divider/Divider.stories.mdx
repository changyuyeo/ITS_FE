# Divider

&lt;Divider /&gt; 컴포넌트 사용법 정리
일직선을 그어주며, 섹션을 나눌 때 유용합니다.

```ts
// Divider 타입 정의
interface DividerProps {
	borderWidth?: number; // default: 1
	color?: 'primary'| 'secondary' | 'success' | 'error' | 'default' | 'description' | 'guide' | 'white'; // default: default
	width?: number;
}
```
<br />

## borderWidth

- `borderWidth` 속성으로 `Divider`컴포넌트의 크기를 변경할 수 있습니다.
