# 아이콘

With Me 아이콘 정리

아이콘을 사용하기 전에 `@with-me/icons 라이브러리`를 먼저 설치해주세요!

> npm install @with-me/icons 또는, <br />
> yarn add @with-me/icons

```js
import { ArrowDown } from '@with-me/icons'
```

위와 같은 방식으로 import 하여 사용하시면 됩니다! <br />
**필요한 아이콘이있으면 요청해주세요!**

<br />
<br />
