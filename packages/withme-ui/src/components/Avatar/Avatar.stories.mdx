# Avatar

&lt;Avatar /&gt; 컴포넌트 사용법 정리

```ts
// Avatar 타입 정의
interface AvatarProps {
	alt?: string; // default: ''
	clickable?: boolean; // default: false
	nickname: string; // required value
	shape?: 'circle' | 'square'; // default: 'circle'
	size?: number | "sm" | "lg"; // default: 'lg'
	src?: string | null; // default: null
}
```
<br />

## 기본 사용법

- `Avatar`컴포넌트는 기본적으로 nickname을 `string 타입`으로 받습니다.
- **src** 속성을 통해 이미지 정보를 받을 수 있습니다. (이미지 src 정보를 입력했다면 `alt` 속성도 입력해주세요!)
- 이미지 정보가 있다면 해당 이미지 보여주며 이미지 정보가 없다면 nickname에 입력했던 내용 중 첫 번째 글씨를 보여줍니다.

- `size` 속성을 이용해서 원하는 크기로 설정할 수 있습니다.
- 크기를 디테일하게 조정하고 싶다면 원하는 수치를 size에 적용하면 됩니다. (px 단위)
