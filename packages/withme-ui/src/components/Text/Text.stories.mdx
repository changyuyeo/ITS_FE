# Text

&lt;Text /&gt; 컴포넌트 사용법 정리

```ts
// Text 타입 정의
interface TextProps {
	color?: 'primary' | 'secondary' | 'success' | 'error' | 'defalut' | 'description' | 'guide' | 'white'; // default: 'default' 
	size?: 'xs' | 'sm' | 'md' | 'base' | 'lg' | 'xl' | 'xxl'; // default: 'base'
	weight?: "medium" | "thin" | "light" | "bold" | "regular"; // default: 'regular'
}
```
<br />
