# Card

&lt;Card /&gt; 컴포넌트 사용법 정리

```ts
// Avatar 타입 정의
interface AvatarProps {
	borderColor?: string; // default: #dddddd`
	borderRadius?: number; // default: 5`
	children: ReactNode; // required value
	fullSize?: boolean; // default: false
	htmlTitle?: string;
	isBorder?: boolean; // default: false
	isHover?: boolean; // default: false
	isShadow?: boolean; // default: false
	px?: number; // default: 20
	py?: number; // default: 20
	title?: ReactNode;
}
```
<br />

## 기본 사용법

- `Card`컴포넌트는 기본적으로 **children**을 받습니다.

<Canvas>
	<Story name="basic">
		<Card>card content</Card>
	</Story>
</Canvas>

<br />

## border

- `borderColor` 속성을 이용하면 Card의 **테두리 색상**을 변경할 수 있습니다.
- `borderRadius` 속성으로 테두리의 **둥글기 정도**를 변경할 수 있습니다.

<Canvas>
	<Story name="borderColor">
		<Card borderColor="#fc5c65">card</Card>
		<Card borderColor="#fd9644">card</Card>
		<Card borderColor="#fed330">card</Card>
		<Card borderColor="#26de81">card</Card>
		<Card borderColor="#45aaf2">card</Card>
		<Card borderColor="#4b7bec">card</Card>
		<Card borderColor="#a55eea">card</Card>
	</Story>
</Canvas>

<Canvas>
	<Story name="borderRadius">
		<Card borderRadius={0}>card</Card>
		<Card borderRadius={5}>card</Card>
		<Card borderRadius={20}>card</Card>
		<Card borderRadius={50}>card</Card>
		<Card borderRadius={999}>card</Card>
	</Story>
</Canvas>

<br />

## fullSize

- `fullSize` 속성을 이용하면 부모너비 기준으로 너비가 100%로 설정됩니다.

<Canvas>
	<Story name="fullSize">
		<Card fullSize>card content</Card>
	</Story>
</Canvas>

<br />

## isBorder

- `isBorder` 속성을 기준으로 Card 컴포넌트의 테두리의 유무를 설정할 수 있습니다.
- `isShadow` 속성으로 Card 컴포넌트의 그림자 효과를 설정할 수 있습니다.

<Canvas>
	<Story name="isBorder">
		<Card isBorder={true}>card with border</Card>
		<Card isBorder={false}>borderless card</Card>
	</Story>
</Canvas>

<Canvas>
	<Story name="isShadow">
		<Card isShadow={true}>card with shadow</Card>
		<Card isShadow={false}>shadowless card</Card>
		<Card isShadow={true} isBorder={false}>borderless card with shadow</Card>
	</Story>
</Canvas>

<br />

## px, py

- `px, py` 속성으로 위아래 혹은 양옆의 패딩을 설정하여 정밀한 크기조정이 가능합니다. (px 단위)

<Canvas>
	<Story name="padding">
		<Card px={100} py={100}>card content (padding 100px)</Card>
	</Story>
</Canvas>

<br />

## title

- `title` 속성은 Card 컴포넌트의 제목을 입력할 수 있는 공간을 제공합니다.

<Canvas>
	<Story name="title">
		<Card title={<Title size="h5">card title</Title>}>card content (padding 100px)</Card>
	</Story>
</Canvas>

<br />
