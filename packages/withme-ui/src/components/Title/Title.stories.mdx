# Title

&lt;Title /&gt; 컴포넌트 사용법 정리

```ts
// Title 타입 정의
interface TitleProps {
	color?: 'primary' | 'secondary' | 'success' | 'error' | 'defalut' | 'description' | 'guide' | 'white'; // default: 'default' 
	size?: 'h1' | 'h2' | 'h3' | 'h4' | 'h5' | 'h6'; // default: 'h2'
	weight?: "medium" | "thin" | "light" | "bold" | "regular"; // default: 'medium'
}
```
<br />

## 기본 사용법

- `Title`컴포넌트는 기본적으로 **children**을 받습니다.

<Canvas>
	<Story name="basic">
		<Title>Title</Title>
	</Story>
</Canvas>

<br />

- `color` 속성을 이용해서 Title 컴포넌트의 색상을 변경할 수 있습니다.
