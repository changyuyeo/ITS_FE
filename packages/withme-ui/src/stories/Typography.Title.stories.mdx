## color

- Title의 색상을 color 속성으로 재설정할 수 있습니다.
- default: 'gray-900'

```ts
type RGB = `rgb(${number}, ${number}, ${number})`;
type RGBA = `rgba(${number}, ${number}, ${number}, ${number})`;
type HEX = `#${string}`;

interface TitleProps {
  color: RGB | RGBA | HEX | 'gray-100' | 'gray-200' | 'gray-300' ... | 'gray-800' | 'gray-900'
  // ...
}
```

## size

- default: 'md'

