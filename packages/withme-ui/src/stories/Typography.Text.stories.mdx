## color

- Text의 색상을 color 속성으로 재설정할 수 있습니다.
- default: 'gray-800'

```ts
type RGB = `rgb(${number}, ${number}, ${number})`;
type RGBA = `rgba(${number}, ${number}, ${number}, ${number})`;
type HEX = `#${string}`;

interface TextProps {
  color: RGB | RGBA | HEX | 'gray-100' | 'gray-200' | 'gray-300' ... | 'gray-800' | 'gray-900'
  // ...
}
```

## size

- Text의 크기를 size 속성을 통해 지정할 수 있습니다.
- default: 'md'

```ts
interface TextProps {
  size?: "xxl" | "xl" | "lg" | "md" | "sm"
  // ...
}
```
