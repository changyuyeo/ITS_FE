# Typography

Typography 컴포넌트 사용법 <br />

- Typography 안에는 Text와 Title 컴포넌트가 있습니다.
- &lt;Text /&gt; 문자열을 나타낼 때 유용합니다.
- &lt;Title /&gt; 제목을 나타낼 때 유용합니다.

<br />

### example
```tsx
import { Typography } from '@with-me/ui'

function App() {
  return (
    <>
      <Typography.Text>Text</Typography.Text>
      <Typography.Title />Title</Typography.Title>
    </>
  )
}

export default App
```
